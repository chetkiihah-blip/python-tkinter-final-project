# python-tkinter-final-project
Fact Tracker — графическое приложение для сбора и использования интересных фактов. Добавляйте факты с категориями, просматривайте список, фильтруйте по категории и получайте случайный факт. Данные хранятся в JSON. Проект написан на чистом Python с разделением логики и интерфейса, включает тесты и документацию.
